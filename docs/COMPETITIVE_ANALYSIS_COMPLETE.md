## 4. KPIs DASHBOARD — Lo que TORQUE 360 debe medir

### 4.1 KPIs Core (Implementar en Dashboard v1)

| KPI | Formula | Benchmark | Alerta |
|-----|---------|-----------|--------|
| **Rotacion inventario** | COGS / Inv. promedio | 6-8x/ano | < 4x = warning |
| **Dias de inventario (DIO)** | 365 / Rotacion | < 45 dias | > 60 dias = alerta |
| **Fill rate** | Ordenes completas / Total | > 95% | < 90% = critico |
| **Stock muerto %** | Items sin venta 12m / Total SKUs | < 5% | > 10% = urgente |
| **Margen bruto por SKU** | (Ingreso - Costo) / Ingreso | 15-25% (mayoreo), 25-40% (retail) | < 10% = revisar |
| **Retencion clientes** | Clientes recurrentes / Total | > 80% (B2B) | < 70% = churn |
| **Valor promedio orden** | Ingresos / Num ordenes | Tracking mensual | Tendencia negativa |
| **Tiempo buscar repuesto** | Busqueda a resultado | < 5 seg | > 15 seg = UX |
| **Precision picking** | Picks correctos / Total | > 99.5% | < 98% = proceso |
| **Devoluciones por proveedor** | Devol/proveedor / Compras | < 2% | > 5% = proveedor |

### 4.2 KPIs Avanzados (Dashboard v2)

| KPI | Descripcion | Impacto |
|-----|-------------|---------|
| **Stockout rate** | % tiempo sin stock de un item | Ventas perdidas |
| **Ciclo OC** | Tiempo desde PO a recepcion | Eficiencia supply chain |
| **On-time delivery proveedor** | % POs entregadas a tiempo | Scorecard proveedor |
| **Ingreso por m2** | Ventas / Area bodega | Eficiencia espacio |
| **Cash-to-cash cycle** | DIO + DSO - DPO | Salud cash flow |
| **SKU profitability ranking** | Top/bottom por margen | Optimizar mix |
| **Cotizacion a venta** | Cotizaciones aceptadas / Total | Efectividad ventas |
| **Precision entrega** | Entregas a tiempo / Total | Satisfaccion cliente |
| **OEE (taller)** | Eficiencia operativa taller | Top performers: 85% |
| **First pass yield** | Reparaciones sin retrabajo / Total | Target > 95% |
